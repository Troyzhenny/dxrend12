# Dxrend 12
dxrend is a minimal DirectX 12 renderer written in C++ built as a learning project to explore the fundamentals of modern graphics programming and low-level GPU APIs.
The goal of this project is to understand how modern rendering pipelines work by implementing systems manually using DirectX 12 and the Win32 API.

### The Goal
My goal with this project is to learn graphics programming, while creating something
i can build on and eventually build my own games with it.

### Technologies Used
- C++20
- DirectX 12
- DXGI
- Win32 API
- DirectXMath
- CMake

### Get started building the project
SN: The executable will be generated in the build directory.
```
git clone https://github.com/yourname/dxrend12
cd dxrend12
mkdir build
cd build
cmake ..
cmake --build .
```
